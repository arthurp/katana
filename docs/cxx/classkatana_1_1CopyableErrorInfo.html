Detailed Description

A CopyableErrorInfo is a variant of ErrorInfo that can be used outside a thread's error stack.

An ErrorInfo is not generally copyable because it references thread local data. A CopyableErrorInfo is useful in cases where one wants to store errors, e.g., collecting results across threads.
