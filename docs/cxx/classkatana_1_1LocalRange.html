Detailed Description

template<typename T>
class katana::LocalRange< T >

A LocalRange is a range specialized to containers that have a concept of local ranges (i.e., local_begin and local_end).

Local ranges partition a container into portions that are local to each thread. The local_begin and local_end methods return the portion local to the current thread.
