Detailed Description

template<typename T>
class katana::LocalTwoLevelRange< T >

A LocalTwoLevelRange is a range over a container (outer) of containers (inner) where the outer container has local ranges and the overall range should be over elements of the inner container.

This range is commonly used when iterating over thread-local containers. The initial elements should be drawn from the container local to the current thread, but overall iteration space is the sum of all thread-local containers (e.g., for work-stealing).
