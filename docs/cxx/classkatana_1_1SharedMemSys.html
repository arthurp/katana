Detailed Description

SharedMemSys initializes the Galois library for shared memory.

Most Galois library operations are only valid during the lifetime of a SharedMemSys or a DistMemSys.

It is not advisable to create a SharedMemSys more than once. Certain downstream implementation dependencies like the AWS SDK cannot be reinitialized.
