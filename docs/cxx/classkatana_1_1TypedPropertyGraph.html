Detailed Description

template<typename NodeProps, typename EdgeProps>
class katana::TypedPropertyGraph< NodeProps, EdgeProps >

A property graph is a graph that has properties associated with its nodes and edges.

A property has a name and value. Its value may be a primitive type, a list of values or a composition of properties.

A TypedPropertyGraph is a representation of a property graph that imposes a typed view on top of an underlying PropertyGraph. A PropertyGraph is appropriate for cases where the graph is largely uninterpreted and can be manipulated as a collection of bits. A TypedPropertyGraph is appropriate for cases where computation needs to be done on the properties themselves.
