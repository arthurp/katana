Detailed Description

A computational plan to for Jaccard, specifying the algorithm and any parameters associated with it.

Member Enumeration Documentation

◆ EdgeSorting

enum katana::analytics::JaccardPlan::EdgeSorting
Enumerator
kUnknown 

The edges may be sorted, but may not.

Jaccard may optimistically use a sorted algorithm and fail over to an unsorted one if unsorted edges are detected.

kSorted 

The edges are known to be sorted by destination.

Use faster sorted intersection algorithm.

kUnsorted 

The edges are known to be unsorted.

Use slower hash-table intersection algorithm.
