Detailed Description

template<typename _IDType, typename _Value>
struct katana::OpaqueIDLinear< _IDType, _Value >

A linear ID.

Linear IDs support:

  • Addition and subtraction of DifferenceType (which is a signed version of ValueType)
  • Increment and decrement
  • Subtraction of two IDs to get a DifferenceType
  • Dereference
