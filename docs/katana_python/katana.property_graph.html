Property Graphs (katana.property_graph)¶

class katana.property_graph.PropertyGraph¶

A property graph loaded into memory.

__init__(self, path, node_properties=None, edge_properties=None, partition_id_to_load=None)¶

Load a property graph.

Parameters

  • path (str) – the path or URL from which to load the graph. This support local paths or s3 URLs.

  • partition_id_to_load – The partition number in the graph to load. If this is None (default), then the partition corresponding to this host’s network id will be loaded.

  • node_properties – A list of node property names to load into memory. If this is None (default), then all properties are loaded.

  • edge_properties – A list of edge property names to load into memory. If this is None (default), then all properties are loaded.

static from_graphml(path, chunk_size)¶

Load a GraphML file into katana form

Parameters

  • path (str or Path) – Path to source GraphML file.

  • chunk_size (int) – Chunk size for in memory representations during conversion. Generally this term can be ignored, but it can be decreased to reduce memory usage when converting large inputs.

Returns

the new PropertyGraph
